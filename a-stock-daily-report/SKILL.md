# A股市场日报技能

## 核心原则
- 收盘后执行（16:00）
- 先总结昨日表现
- 提醒用户提供股票代码
- 生成完整简报

## 执行步骤（严格按顺序执行）

1. **读取配置**
   - 读取 ~/a-stock-trader/today_stocks.txt
   - 检查b1、b2是否有值

2. **判断执行分支**
   - 如果b2为空 → 发送提醒："请提供今日b2股票代码"
   - 如果两只股票都有 → 执行第3步

3. **生成简报**
   - 运行：`python3 ~/a-stock-trader/stock_analyzer.py <b1> <b2>`
   - 获取昨日表现、今日涨跌、趋势信号

4. **发送到飞书**
   - 使用feishu_doc发送报告

## 输出格式

**提醒格式**：
```
📊 A股日报 - 请提供股票代码

请提供今日b2股票代码，完成配置后明天的日报将包含完整分析。
```

**完整简报格式**：
```
📊 A股市场日报 - 2026-02-16

📈 上证指数: 3350.00 (+0.25%)
📊 股票1: 贵州茅台 1850.00 (+1.20%) | 趋势: 上涨 | 信号: 买入
📊 股票2: 宁德时代 420.50 (-0.80%) | 趋势: 下跌 | 信号: 持有

💡 今日观察：xxx
```
