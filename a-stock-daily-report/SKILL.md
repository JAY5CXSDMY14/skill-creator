# A股市场日报技能

## 核心原则
- 收盘后执行（16:00）
- 先总结昨日表现
- 提醒用户提供股票代码
- 生成完整简报

## 执行步骤（严格按顺序执行）

1. **读取配置**
   - 读取 ~/a-stock-trader/today_stocks.txt
   - 检查b1、b2是否有值

2. **判断执行分支**
   - 如果b2为空 → 发送提醒
   - 如果两只股票都有 → 执行第3步

3. **生成简报**
   - 运行：`python3 ~/a-stock-trader/stock_analyzer.py <b1> <b2>`
   - 获取：支撑位、压力位、MA20、趋势
   - 计算b1（KDJ J值大负值）、b2（第一根中长阳）

4. **发送到飞书**
   - 使用feishu_doc发送报告
